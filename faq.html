Question-1: What are the different ways to select an element in the DOM?

We can apply 5 ways to select an element in the DOM.
1- getElementById().
2-getElementsByClassName().
3-getElementsByTagName().
4-querySelector().
5-querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

In simple word, innerHTML sort out the full HTML content within an element along with all tags, while "innerText" offers only the visible text content of an element, further "textContent" sort out all text content within an element,like hidden text and whitespace, but no chance of interpreting HTML tags.

Question-3: What is event delegation in the DOM?

In this event delegation technique you can attach a single event listener to a parent element to handle it's child elements, ne need of adding separate listeners to each child element.

Question-4: What is event bubbling in the DOM?

Event Bubbling occurs while an element gets an event, and this event bubbles up to its ancestor & parent elements in the DOM tree until it touch the root element.

Question-5: How do you create, add, and remove elements using JavaScript?

To create a new element in HTML apply-
document.createElement().

To add a new element in HTML apply-
document.appendChild().

To remove existing element in HTML apply-
